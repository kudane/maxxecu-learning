* หัวข้อหลัก: Idle Angle Mode
* วิธีควบคุมรอบเดินเบาด้วยองศาไฟจุดระเบิด (Ignition advance) โดยมีโหมด Disabled เพื่อเลือกไม่ใช้ระบบนี้
* มีโหมด Idle ignition locked / by separate table สำหรับล็อคค่าองศาไฟหรือใช้ตารางแยก และโหมด Idle ignition controlled by idle system เพื่อปรับไฟอัตโนมัติ
* หัวข้อหลัก: Ignition Source
* แหล่งกำหนดองศาไฟตอน idle เลือกได้ระหว่าง Fixed value (ใช้ค่าคงที่) หรือ Table (ใช้ตารางกำหนดตามเงื่อนไข)
* หัวข้อหลัก: Idle Ignition Lock
* Idle ign angle คือค่าองศาไฟที่ระบบจะนำไปใช้งานเมื่อเลือกแหล่งกำหนดเป็นแบบ fixed value
* ข้อกำหนดเพิ่มเติมกรณีใช้แบบ table คือช่องขวาสุดของตารางต้องตั้งค่าเป็น 0° เสมอ
* หัวข้อหลัก: เงื่อนไขปิดการทำงาน
* ระบบจะปิดการทำงานทันทีเมื่อค่าคันเร่ง คลื่นโหลด หรือความเร็ว เกินกว่าค่ากำหนดใน Max TPS, Max MAP และ Max speed
* ฟังก์ชัน Max RPM ใช้สำหรับสั่งปิดระบบควบคุมองศาไฟเดินเบาเมื่อรอบเครื่องยนต์เกินค่าที่ตั้งไว้

```ตัวอย่าง
- ค่าปกติของ Max RPM จะตั้งไว้ที่ประมาณ ~2000–2500 rpm

```

* หัวข้อหลัก: ช่วงการควบคุม Ignition
* Idle control max retard angle คือมุมไฟต่ำสุดที่ระบบสามารถลดได้
* Idle control max advance angle คือมุมไฟสูงสุดที่ระบบสามารถเพิ่มได้

```ตัวอย่าง
- Idle control max retard angle สัมพันธ์กับ 0% duty
- Idle control max advance angle สัมพันธ์ com กับ 100% duty

```

* หัวข้อหลัก: การใช้ Correction อื่นร่วม
* Apply adjustments when idle ign active ใช้กำหนดว่าจะให้ค่า correction อื่นมีผลร่วมด้วยตอน idle หรือไม่
* โหมด Enabled จะยินยอมให้ใช้ระบบชดเชยจากอุณหภูมิและอากาศ ส่วน Disabled จะสั่งไม่ใช้ค่า correction ใด ๆ

```ตัวอย่าง
- ค่า correction อื่น เช่น CLT และ IAT

```

* หัวข้อหลัก: หมายเหตุสำคัญ
* ฟังก์ชัน Anti-stall (เก่า) ถูกถอดออกจากระบบแล้ว โดยข้อแนะนำกำหนดให้เปลี่ยนไปใช้งาน ignition idle control แทน
* หัวข้อหลัก: อธิบายแบบใช้งานจริง - หลักการ
* ECU จะใช้ไฟจุดระเบิดเป็นตัวปรับรอบเครื่องยนต์โดยตรง โดยอาศัยการเพิ่มลด torque จาก ignition แทนการเพิ่มลม
* มี Logic การทำงานคือ ไฟอ่อน (retard) ทำให้ torque ลดลงส่งผลให้รอบตก และไฟแก่ (advance) ทำให้ torque เพิ่มขึ้นส่งผลให้รอบขึ้น
* หัวข้อหลัก: อธิบายแบบใช้งานจริง - ตัวอย่างจริง
* พฤติกรรมการดันและดึงองศาไฟของ ECU เมื่อกำหนดรอบเป้าหมายและช่วงองศาไฟจุดระเบิด

```ตัวอย่าง
- ตั้งช่วง ignition: 4° → 36°
- Target idle: 1000 rpm
- ถ้ารอบตก: ECU จะเพิ่มไฟไป 20° → 30° เพื่อให้รอบขึ้น
- ถ้ารอบสูง: ECU จะลดไฟลง → 10° → 5° เพื่อให้รอบลง

```

* หัวข้อหลัก: อธิบายแบบใช้งานจริง - ข้อดี / ข้อเสีย
* ข้อดีคือระบบตอบสนองเร็วมาก เหมาะกับการทำ fine tuning รอบนิ่งๆ และประยุกต์ใช้ร่วมกับระบบ dual mode ได้ดี
* ข้อเสียคือถ้ากำหนด range แคบจะคุมไม่อยู่ ถ้ากว้างเกินรอบจะแกว่งแรง และถ้า base ignition map มั่ว ระบบ idle จะเพี้ยนทันที
* หัวข้อหลัก: อธิบายแบบใช้งานจริง - Insight ที่คนพลาดกันเยอะ
* ระบบระบุให้ Ignition ทำหน้าที่ปรับละเอียด ส่วน Air (IAC / throttle) ทำหน้าที่ปรับหยาบ
* ในระบบ Single mode ignition ต้องตั้ง base airflow ให้พอดี ส่วนระบบ Dual mode ไฟจุดระเบิดจะทำงานก่อนแล้ว air ค่อยตาม
* หัวข้อหลัก: สรุปสั้น
* Ignition idle control คือการใช้ไฟดันหรือดึงรอบเครื่องยนต์ และต้องกำหนดช่วงองศาปรับ min/max ให้เหมาะสม
* มีเงื่อนไขการปิดระบบเช่นเดียวกับ closed loop และให้ผลลัพธ์ดีที่สุดเมื่อนำไป combine ร่วมกับ air control (dual mode)

```แจ้งเตือน
- รายละเอียดเงื่อนไขและวิธีตั้งค่าในตาราง (Table) ของฝั่ง Ignition Source [ไม่มีในแหล่งที่มา]
- คำศัพท์ BTDC, Feed Forward, PWM และ PID ปรากฏในเงื่อนไขและข้อบังคับหลัก แต่ไม่ถูกกล่าวถึงในเนื้อหาชุดนี้ [ไม่มีในแหล่งที่มา]

```
