### **MaxxECU Idle Control Target RPM (ระบบ Idle)**

#### **Target RPM Settings**

* **Idle RPM target table:** ตารางกำหนด “รอบเป้าหมาย” ที่ระบบจะพยายามรักษาไว้ในทุกสถานการณ์ของ idle เช่น ตามอุณหภูมิ
* **AC idle up RPM:** ค่ารอบที่จะ “เพิ่มขึ้น” เมื่อมีการสั่งงานแอร์หรือคอมแอร์ทำงาน
* **Clutch idle up RPM:** ค่ารอบที่จะ “เพิ่มขึ้น” เมื่อมีสัญญาณคลัทช์ เช่น เหยียบคลัทช์

#### **อธิบายแบบใช้งานจริง**

##### **1. Idle RPM Target Table**

* **แนวคิดสำคัญ:** เป็น “เป้าหมายหลัก” ที่ระบบ Closed Loop จะพยายามดันให้รอบเครื่องยนต์ขึ้นไปถึงจุดที่กำหนดไว้
* **แนวคิดสำคัญ:** ถูกนำไปใช้ควบคุมในทุกกรณี เช่น เดินเบาปกติ, หลังถอนคันเร่ง, และตอนมีโหลดเพิ่ม

```
ตัวอย่าง:
| Temp | Target RPM |
|---|---|
| 20°C | 1100 |
| 60°C | 900 |
| 90°C | 800 |
- เครื่องเย็น → ต้องรอบสูง (กันดับ)
- เครื่องร้อน → ลดรอบให้นิ่ง

```

##### **2. AC Idle Up RPM**

* **แนวคิดสำคัญ:** เมื่อแอร์ทำงานเครื่องยนต์จะโดนโหลดเพิ่ม ระบบ ECU จะทำการ “เพิ่ม target RPM” ขึ้นชั่วคราว
* **แนวคิดสำคัญ:** หากไม่ตั้งค่านี้ เมื่อเปิดแอร์รอบเครื่องยนต์จะวูบ หรือเกิดอาการเครื่องยนต์ดับลงทันที

```
ตัวอย่าง:
- Target ปกติ = 800 rpm  
- AC idle up = +100 rpm  
→ รอบเป้าหมายใหม่กลายเป็น 900 rpm

```

##### **3. Clutch Idle Up RPM**

* **แนวคิดสำคัญ:** ใช้ในจังหวะ “เหยียบคลัทช์ / เข้าเกียร์ว่าง” เพื่อป้องกันรอบตกในตอนที่โหลดหายไปอย่างรวดเร็ว
* **แนวคิดสำคัญ:** เหมาะสำหรับรถเกียร์ธรรมดา และรถที่รอบตกไวในจังหวะเปลี่ยนเกียร์

```
ตัวอย่าง:
- ตอนวิ่งแล้วเหยียบคลัทช์ รอบจะไม่ร่วงฮวบแต่ถูก “พยุง” ไว้
- ค่าตัวอย่างจริงจากวิดีโอฝึกอบรม: ตั้งค่าชดเชยเพิ่มขึ้น 25 RPM หรือ 50 RPM [จากวิดีโอฝึกอบรม Part 34]

```

#### **ความสัมพันธ์กับ Closed Loop**

* **แนวคิดสำคัญ:** Target RPM ทำหน้าที่เป็น “เป้า” ส่วน Closed Loop คือ “คนพยายามยิงให้โดนเป้า”
* **แนวคิดสำคัญ:** ถ้า Target ตั้งมั่ว Closed Loop จะมั่วตาม และถ้า Open Loop ไม่ใกล้เคียง จะทำให้ Closed Loop ทำงานหนักจนรอบแกว่ง

```
Logic การทำงานจริง:
1. อ่าน Target RPM จาก table  
2. เทียบกับ RPM จริง  
3. เอา error ไปคำนวณ (PID)  
4. สั่ง “ลม / ไฟ” ให้เข้าใกล้เป้า

```

#### **ข้อผิดพลาดที่เจอบ่อย**

* **แนวคิดสำคัญ:** การตั้ง Target ต่ำเกินไปทำให้รถสั่นหรือดับ ส่วนการตั้งสูงเกินไปทำให้กินน้ำมันและเสียงเครื่องยนต์สูงผิดปกติ
* **แนวคิดสำคัญ:** ไม่ใส่ AC idle up ทำให้เปิดแอร์แล้วรอบตก และถ้าค่า Open Loop ไม่ตรง จะทำให้ระบบ Closed Loop ไล่รอบไม่ทัน
