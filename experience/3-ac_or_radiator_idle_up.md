* หัวข้อหลัก: สรุปความแตกต่างในการตั้งค่า
* AC Idle Up ทำหน้าที่เพิ่มทั้งรอบเป้าหมาย (RPM) และบวกค่า Duty Cycle เพิ่มเข้าไปในระบบ
* Radiator Fan Idle Up จะเน้นบวกลบเฉพาะค่า Duty Cycle (%) เพิ่มเข้าไปเพื่อทำระบบ Load Compensation ชดเชยโหลดไฟฟ้า
* หัวข้อหลัก: วิธีการใส่ค่า: Interpolation vs. Lock Duty
* ช่วงเครื่องเย็น (Cold Start) ควรปล่อยให้ระบบทำการ Interpolation ลาดลงตามอุณหภูมิที่เปลี่ยนแปลงไป
* ช่วงเครื่องร้อน (Operating Temp) ควรใช้การ Lock Duty หรือใส่ค่าคงที่เท่ากันในย่านอุณหภูมิทำงาน

```ตัวอย่าง
- ช่วงเครื่องร้อน (80-95°C) → หากใส่ค่า 80°C = 18% และ 90°C = 17% แล้วพัดลมทำงานที่ 85°C ค่าฐานจะเปลี่ยนเป็น 17.5% ทันที ทำให้จูนค่า Offset (3%) ได้ยากเพราะค่าตั้งต้นขยับไปมา

```

```แจ้งเตือน
- คำศัพท์เทคนิค BTDC, Feed Forward, PWM และ PID มีการระบุให้คงไว้ในกฎ แต่ไม่ถูกกล่าวถึงในเนื้อหาต้นฉบับชุดนี้ [ไม่มีในแหล่งที่มา]

```
